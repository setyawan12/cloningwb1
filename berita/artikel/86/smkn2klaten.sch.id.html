PENDAFTARAN PESERTA DIDIK BARU TAHUN 2014/2015

SMK NEGERI 2 KLATEN PROGRAM DIKLAT 4 TAHUN

I. Program Keahlian Yang Dibuka dan Daya Tampung :

  1. Teknik Konstruksi Bangunan/Sipil ( TKB / 1 kelas )
  2. Teknik Audio Video ( TAV / 2 kelas )
  3. Teknik Instalasi Tenaga Listrik ( TITL / 2 kelas )
  4. Teknik Pemesinan ( TPm / 2 kelas )
  5. Teknik Pengecoran Logam ( TPL / 1 kelas )
  6. Teknik Kendaraan Ringan / Otomotif ( TKR / 2 kelas )
  7. Teknik Komputer Jaringan ( TKJ / 2 kelas )
  8. Teknik Gambar Bangunan ( TGB /2 kelas )

• Jumlah siswa per klas : 32 siswa

II. Waktu dan Tempat Pendaftaran :
Pendaftaran online dilaksanakan bulan Juni 2014 ( Menunggu surat resmi dari Dinas Pendidikan Kabupaten Klaten ).

III.Persyaratan Umum Pendaftaran :

  1. Calon peserta didik telas Lulus dan memiliki ijazah SMP/MTs/Program Paket B dan memiliki ( STL / SKHUN ):
  2. Lulus tes kesehatan oleh dokter yang ditunjuk oleh SMK Negeri 2 Klaten. 
  3. Mengikuti Tes Khusus Tertulis ( akademis dan psikotes ).
  4. Setinggi-tingginya berusia 21 ( dua puluh satu ) tahun pada tanggal 14 Juli 2014.
  5. Melakukan pendaftaran secara online yang dilakukan oleh operator sekolah.
  6. Pendaftaran di SMK Negeri 2 Klaten dapat memilih 2 pilihan program keahlian / jurusan.
  7. Calon siswa dari luar kabupaten maksimal 10 %

IV. Program Unggulan Sekolah :

  1. Pendidikan menerapkan Sistem Managemen Mutu ISO. 9001:2008;
  2. Terpilih sebagai SMK Rujukan di kabupaten Klaten , salah satu SMK dari sepuluh SMK di Indonesia;
  3. Program BMW : Bekerja : setelah lulus yang ingin bekerja disalurkan Melanjutkan : Program bimbingan belajar intensif bagi yang melanjutkan ke perguruan tinggi: Wirausaha : Program intensif menyiapkan wirausahaan yang tangguh:
  4. Beasiswa bagi siswa berprestasi dan kurang mampu;
  5. Prakerin dan magang kerja pada industri berstandar internasional.

V. Informasi lebih lanjut :
Hubungi : 0272 3354021 atau lihat Webside : Smkn2klaten.sch.id Klaten, 01 Mei 2014 Kepala Sekolah Drs. Wardani Sugiyanto, MPd NIP. 19640311 198910 1 001
